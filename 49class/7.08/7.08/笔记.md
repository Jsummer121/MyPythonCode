### 16-linux+vim

#### 1.linux命令

```
进程：简单理解为正在进行中的程序。
```

```
top：进程监视，监视系统的实时状态
ps：提供进程的一次性查看

ps -ef |grep 8000
kill 进程号

workon 	# 查看虚拟环境名
workon 环境名 # 进入指定的虚拟环境
mkvirtualenv -p /usr/bin/python 环境名	# 创建一个新的环境
```

#### 2.vim（快捷操作方式）

```
vim 文件名
python 文件名
```

```
vim三种模式：命令模式、插入模式、末行模式
插入模式、末行模式---->命令模式：esc
命令模式---->末行模式：用冒号：
命令模式---->插入模式：用（iIaAoO）
```

```
vim 文件名
此时在命令模式---->进入编辑状态---->用（iIaAoO）进入插入模式---->编写完毕，需要保存退出---->需要进入末行模式（esc进入命令模式：进入末行模式wq保存退出）
q！：强制退出
shift+zz(命令模式) 保存并退出
```

```
插入模式：
i	从原光标位置前插入
I	从原光标所在行的行首插入
a	从原光标位置后插入
A	从原光标所在行的行尾插入
o	原光标所在行的下方新增一行并且进入插入模式
O	原光标所在行的上方新增一行并且进入插入模式

esc退出 插入模式 进入命令模式

命令模式：
光标的移动
1.H左J下K上L右
2.gg 到文件第一行的行首 G 到文件最后一行的行首
3.0 到当前行的行首
4.^ 当前非空格行首 $ 当前非空格行尾

以上都是基本操作。
1.yy 复制整行 3yy 复制3行内容  yw 复制光标所在位置的单词
2.p 粘贴
3.dd 删除整行（剪切，p粘贴） 5dd 删除5行 dw删除光标所在位置的单词 x 删除光标所在的字符
4.u 撤销 ctrl+r 反撤销
5.块选择：v （行首) V (整行) ctrl+v 列块选择
6.shift+>：向右缩进；shift+< 往左缩进

命令模式通过：进入末行模式：
末行模式：
/搜索内容	n向下，N向上
？搜索内容	n向上，N向下

n，N（向上向下查找）


替换：%s/原/新/g
```
