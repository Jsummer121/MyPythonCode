# 内建函数

我们知道python中一切皆对象，并且像python中print函数和input函数等都是哪里来的呢？

我们来看一下，在python中有一个globals函数，它可以打印出当前文件中所有的全局变量，我们来看个例子：

```python
# -*- coding: utf-8 -*-
# @Author  : summer


a = 100


def b():
	pass


class C:
	pass


g = globals()
print(g)
```
