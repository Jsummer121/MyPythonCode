### 01-基本数据类型一

#### 一、查看数据类型的方法：

python基本数据类型包括了：数字类型、布尔值、字符串、列表、元组、字典、集合

##### 1.type：查看数据类型

```
temp = 'hello'
t = type(temp)
print(t)
# Ctrl+鼠标左键单击str，找到str类及内部的所有方法
```

##### 2.help，type：查看数据类型下的全部方法

```
help(type(temp))
```

##### 3.dir：查看类下全部方法的名字

```
d = dir(temp)
print(d)
```

##### 4.查看类下某特定方法

```
temp.upper()	# Ctrl+鼠标左键单击upper（），自动定位到upper（）方法
```
